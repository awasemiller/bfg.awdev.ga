Introduction

This document contains information on how to perform an upgrade of SGS 0.9 to SGS 1.0.

Upgrade

Backup custom templates and modules.
  • Backup any custom modules you have created.
  • Backup any custom templates you have created.
Remove old files
  • remove all files from the _include/ directory
  • remove all files from the _cache/ directory

Once the files and folders above have been removed, perform the install process as described in the install documentation as if you were deploying your site for the first time.

Post Upgrade

  • To test your custom SGS 0.9 Template on SGS.1.0 update the $sgs_version variable in your templates functions.php. [ $sgs_version = '1.0'; ]
    This will allow you to select your custom template in the administration area.
    You can now make changes to your template code if required.
    Please read the changelog for any changes to SGS 1.0 that may effect your template.
